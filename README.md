# ☁️ CloudBus Pass – Smart Bus Pass Management System

## 📌 Overview

**CloudBus Pass** is a modern web-based bus pass management and ticket booking system that demonstrates how cloud technologies can streamline public transportation services. The application enables passengers to book bus passes, select seats, generate secure QR-based tickets, browse routes, and monitor cloud infrastructure through an interactive dashboard.

Designed with a clean, responsive interface, CloudBus Pass showcases cloud-enabled transportation concepts including secure ticketing, real-time infrastructure monitoring, auto-scaling simulation, and distributed system architecture.
